2026年、ソフトウェア開発における「グリーン・コーディング」は単なるトレンドから、企業が優先すべき必須の非機能要件へと進化しました。本記事では、ドメイン駆動設計（DDD）を用いてシステムの環境負荷を可視化し、Claude CodeのAIコーディング支援を活用して、持続可能なアーキテクチャを構築する方法を解説します。

## 1. 持続可能性のためのDDD（Domain-Driven Design for Sustainability）

従来のDDDでは「ビジネスドメインの複雑さ」を解決することに焦点が当てられてきましたが、2026年のアーキテクチャ設計では**「環境負荷（Carbon Footprint）」をファーストクラスのドメイン知識として扱う**必要があります。

具体的には、以下の戦略をとります：
*   **サステナビリティ・境界づけられたコンテキスト**: サーバーの電力消費やデータ転送量をドメインモデルの一部として定義する。
*   **イベント駆動による最適化**: 非同期処理を駆使し、電力供給が安定している時間帯や、カーボン強度が低いリージョンへ処理をオフロードする。

---

## 2. 実装：`src/domain/sustainability_metrics.ts`

まずは、環境負荷を測定するためのドメインモデルを定義します。Claude Codeに以下の要件を伝え、実装を生成させました。

```typescript
// src/domain/sustainability_metrics.ts

export interface CarbonFootprint {
  kwhConsumed: number;
  carbonIntensityGramsPerKwh: number;
  timestamp: Date;
}

export class SustainabilityMetrics {
  constructor(
    public readonly id: string,
    private metrics: CarbonFootprint
  ) {}

  public calculateTotalEmissions(): number {
    return this.metrics.kwhConsumed * this.metrics.carbonIntensityGramsPerKwh;
  }

  public isWithinThreshold(limitGrams: number): boolean {
    return this.calculateTotalEmissions() <= limitGrams;
  }
}
```

このモデルをエンティティとして扱うことで、ビジネスロジック内で「この処理を実行するとどれだけのCO2が排出されるか」を判定可能にします。

## 3. 環境負荷可視化CLIツール

Claude Codeを活用し、現在のシステム全体の環境負荷をリアルタイムで可視化するCLIツールを構築します。

```typescript
// src/cli/monitor.ts
import { Command } from 'commander';
import { SustainabilityMetrics } from '../domain/sustainability_metrics';

const program = new Command();

program
  .version('1.0.0')
  .description('環境負荷モニタリングツール')
  .action(async () => {
    // 実際にはAPIからメトリクスを取得
    const currentUsage = new SustainabilityMetrics('node-01', {
      kwhConsumed: 0.5,
      carbonIntensityGramsPerKwh: 450,
      timestamp: new Date(),
    });

    console.log(`現在の排出量: ${currentUsage.calculateTotalEmissions()}g CO2`);
    
    if (!currentUsage.isWithinThreshold(200)) {
      console.warn('⚠️ 警告: 排出量が閾値を超えています。負荷分散を検討してください。');
    }
  });

program.parse(process.argv);
```

Claude Codeに「このCLIツールをPrometheusメトリクスと連携させるようにリファクタリングして」と指示するだけで、インフラ層との統合も数分で完了します。

IoTデバイスで取得した温度、湿度、CO2濃度などの環境データを、サーバーレスかつ爆速で公開したいと考えたことはありませんか？

本記事では、Cloudflareの新しいファイル共有サービス「Cloudflare Drop」を活用し、Pythonスクリプトで環境データを自動更新するダッシュボードの構築手法を解説します。

---

## 1. Cloudflare Dropとは？
Cloudflare Dropは、セキュアで高速なファイルアップロード・共有のためのサービスです。従来のような複雑なサーバー構築やFTP設定は不要。Cloudflareのグローバルネットワーク上にファイルを配置し、URLを共有するだけで即座にコンテンツを公開できます。

---

## 2. Pythonによるデータプッシュの実装
センサーから取得したデータをJSON形式にまとめ、Cloudflare DropのAPI（またはアップロードエンドポイント）へ送信するスクリプト例です。

```python
import json
import requests
from datetime import datetime

# センサーデータのシミュレーション
def get_sensor_data():
    return {
        "timestamp": datetime.now().isoformat(),
        "temperature": 24.5,
        "humidity": 45.2,
        "co2_ppm": 420
    }

def upload_to_drop(data):
    # Cloudflare DropのアップロードURL（発行されたURL）
    drop_url = "https://your-drop-url.cloudflare.com/upload"
    
    payload = json.dumps(data)
    headers = {'Content-Type': 'application/json'}
    
    response = requests.post(drop_url, data=payload, headers=headers)
    
    if response.status_code == 200:
        print("データ更新成功")
    else:
        print(f"エラー: {response.status_code}")

if __name__ == "__main__":
    data = get_sensor_data()
    upload_to_drop(data)
```

---

## 3. 可視化ダッシュボードの構築ステップ

1. **Cloudflare Dropの作成**: CloudflareダッシュボードからDropを作成し、一意のURLを取得します。
2. **フロントエンドの準備**: `index.html`を作成し、`fetch()`関数を使ってDrop上のJSONファイルを定期的に読み込むJavaScriptを記述します。
3. **可視化ライブラリの導入**: [Chart.js](https://www.chartjs.org/) を利用すると、数行のコードでグラフ化が可能です。
4. **自動実行環境**: Raspberry PiやGitHub ActionsのScheduled Workflowsを使用して、上記のPythonスクリプトを5分おきに実行するように設定します。

これで、特別なサーバー費用をかけずに、世界中どこからでも閲覧可能な環境モニタリングシステムが完成します。

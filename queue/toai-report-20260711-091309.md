気候変動や環境負荷が可視化される現代において、環境データをリアルタイムで追跡するダッシュボードの需要が高まっています。しかし、インフラの構築に時間をかけていては、刻々と変化する環境データに即応できません。

本記事では、**Cloudflare Pages**を活用し、Pythonで生成した環境データを即座に公開する「レジリエンス・ダッシュボード」の構築方法を解説します。

---

## 1. Cloudflare Pagesによる即時公開のメリット

Cloudflare Pagesは、静的サイトを世界中のエッジネットワークから高速配信するためのプラットフォームです。

*   **即時デプロイ:** GitHubリポジトリと連携し、プッシュするだけで自動ビルド・公開されます。
*   **グローバルCDN:** 世界中のデータセンターから配信されるため、アクセスが非常に高速です。
*   **無料枠の充実:** 個人のプロジェクトであれば、十分な帯域とビルド回数が無料で提供されます。

## 2. Pythonによる環境データ取得とダッシュボード生成

今回は、OpenWeatherMap APIを利用して気温や湿度を取得し、`pandas`で集計してHTMLファイルを生成するスクリプトを作成します。

### 事前準備
```bash
pip install requests pandas
```

### `generate_dashboard.py`
```python
import requests
import pandas as pd
from datetime import datetime

# API設定（OpenWeatherMap等のAPIキーを取得してください）
API_KEY = "YOUR_API_KEY"
CITY = "Tokyo"
URL = f"http://api.openweathermap.org/data/2.5/weather?q={CITY}&appid={API_KEY}&units=metric"

def fetch_data():
    response = requests.get(URL).json()
    data = {
        "timestamp": [datetime.now().strftime("%Y-%m-%d %H:%M:%S")],
        "city": [response["name"]],
        "temp": [response["main"]["temp"]],
        "humidity": [response["main"]["humidity"]]
    }
    return pd.DataFrame(data)

# データ取得とHTML生成
df = fetch_data()
html_content = f"""
<html>
<body>
    <h1>地球環境レジリエンス・ダッシュボード</h1>
    <p>更新日時: {df['timestamp'][0]}</p>
    <table border="1">
        <tr><th>都市</th><th>気温(℃)</th><th>湿度(%)</th></tr>
        <tr><td>{df['city'][0]}</td><td>{df['temp'][0]}</td><td>{df['humidity'][0]}</td></tr>
    </table>
</body>
</html>
"""

with open("index.html", "w", encoding="utf-8") as f:
    f.write(html_content)
```

## 3. CI/CDパイプラインの構築ステップ

GitHub Actionsを使用して、定期的にデータを更新してデプロイするパイプラインを構築します。

1.  **GitHubリポジトリの作成:** プロジェクトをGitHubにプッシュします。
2.  **Cloudflare Pagesの設定:** 
    *   Cloudflareダッシュボードから「Pages」→「プロジェクトを作成」→「Gitに接続」を選択。
    *   リポジトリを選択し、ビルド設定で「Build command」を `python generate_dashboard.py`、「Output directory」を `.` に設定します。
3.  **GitHub Actionsの追加:** `.github/workflows/deploy.yml` を作成し、定期実行（Cron）を設定します。

```yaml
name: Deploy Dashboard
on:
  schedule:
    - cron: '0 * * * *' # 1時間ごとに実行
  push:
    branches: [main]
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Setup Python
        uses: actions/setup-python@v4
        with: {python-version: '3.9'}
      - run: pip install requests pandas
      - run: python generate_dashboard.py
      - name: Publish to Cloudflare Pages
        uses: cloudflare/pages-action@v1
        with:
          apiToken: ${{ secrets.CLOUDFLARE_API_TOKEN }}
          accountId: ${{ secrets.CLOUDFLARE_ACCOUNT_ID }}
          projectName: 'your-project-name'
          directory: '.'
```

この設定により、環境データが自動的に更新され、常に最新のレジリエンス状況が世界中に公開されます。

# Cloudflareだけで独自ドメインメールを構築し、地球環境監視ログを低コストで運用する技術

地球環境の監視データは、膨大なログを低コストかつ高信頼性で保持する必要があります。本記事では、Cloudflareのインフラのみを活用し、独自ドメインメールの運用と監視ログの収集を実現するアーキテクチャを解説します。

## 1. Cloudflare Email Routingによるメール基盤
独自ドメインを用いた通知メールシステムを、サーバーを立てずに構築します。Cloudflare Email Routingを利用することで、受信した環境監視アラートをWorkerに転送し、トリガーに応じた処理を実行可能です。

## 2. Cloudflare Workers + D1によるログ保存
監視ログはCloudflare D1に蓄積します。サーバーレスで完結するため、運用コストはほぼゼロです。

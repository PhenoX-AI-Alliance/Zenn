# はじめに
環境保護団体にとって、セキュアで信頼性の高い通信基盤は活動の根幹です。本記事では、Cloudflare Email Routingを活用し、低コストかつ堅牢な独自ドメインメール環境を構築する方法を解説します。

# Cloudflare Email Routingの設定手順
1. **ドメインの追加**: Cloudflareダッシュボードでドメインを登録します。
2. **DNS設定**: MXレコードとTXTレコード（SPF/DKIM/DMARC）を適切に設定し、なりすましを防ぎます。
3. **ルーティング設定**: `Email Routing`タブから、受信したいメールアドレスを定義し、転送先（Gmail等）を指定します。
4. **セキュリティ強化**: Cloudflareのスパムフィルタリング機能を有効化し、不要な通信を遮断します。

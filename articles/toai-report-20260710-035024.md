# 概要
本記事では、`/src/ddd_flow/` 配下のコードベースに対し、Claude Codeを用いて設計の最適化を行い、計算資源の消費を抑えるDDD（ドメイン駆動設計）フローを解説します。

# 手順
1. Claude Codeをインストールし、プロジェクトルートで起動します。
2. `claude ask "Refactor the domain models in /src/ddd_flow/ to minimize memory footprint and optimize execution path."` を実行します。
3. 生成された最適化済みコードを検証します。

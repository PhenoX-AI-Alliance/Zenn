- **サーバーレスアーキテクチャ**：使用しない時間帯のサーバーを自動停止（例：AWS LambdaやGoogle Cloud Functions）。

## コード例：関数のエネルギー消費を計測するPythonスクリプト

以下は、`codecarbon`ライブラリを用いて関数のCO₂排出量を計測する例です：

```python
from codecarbon import EmissionsTracker
import time

def measure_energy(func):
    def wrapper(*args, **kwargs):
        tracker = EmissionsTracker()
        tracker.start()
        result = func(*args, **kwargs)
        emissions = tracker.stop()
        print(f"CO₂排出量: {emissions:.4f} kg")
        return result
    return wrapper

@measure_energy
def ai_model_training():
    # 模擬的なAI学習処理
    time.sleep(2)
    return "学習完了"

if __name__ == "__main__":
    ai_model_training()
```

**実行方法**：
1. `pip install codecarbon` でライブラリをインストール
2. 関数にデコレータを適用し、実行時にCO₂排出量を出力

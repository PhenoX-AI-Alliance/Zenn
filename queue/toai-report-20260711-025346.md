環境データの可視化は、レジリエンス（回復力）を高めるための第一歩です。しかし、本格的なサーバー構築は手間がかかります。今回は、サーバーレスで静的ファイルを即時公開できる「Cloudflare Drop」を活用し、環境モニタリングダッシュボードを構築する方法を解説します。

---

## 1. Cloudflare Dropとは？
Cloudflare Dropは、ドラッグ＆ドロップでファイルをアップロードするだけで、即座にグローバルCDN経由で公開できるサービスです。複雑な設定やビルドパイプラインなしで、静的サイトを世界中に配信できます。

## 2. 環境データ取得用Pythonスクリプト
まずは、OpenWeatherMap APIなどからデータを取得し、JSONとして保存するスクリプト例です。

```python
import requests
import json

# API設定
API_KEY = "YOUR_API_KEY"
CITY = "Tokyo"
URL = f"http://api.openweathermap.org/data/2.5/weather?q={CITY}&appid={API_KEY}&units=metric"

def fetch_data():
    response = requests.get(URL)
    if response.status_code == 200:
        data = response.json()
        # 必要なデータのみ抽出
        env_data = {
            "city": data["name"],
            "temp": data["main"]["temp"],
            "humidity": data["main"]["humidity"],
            "description": data["weather"][0]["description"]
        }
        with open("data.json", "w") as f:
            json.dump(env_data, f)
        print("Data updated.")

if __name__ == "__main__":
    fetch_data()
```

## 3. ダッシュボードのHTML/JS
`index.html`を作成し、`data.json`を読み込んで表示します。

```html
<!DOCTYPE html>
<html>
<body>
    <h1>環境レジリエンス・ダッシュボード</h1>
    <div id="dashboard">読み込み中...</div>

    <script>
        fetch('data.json')
            .then(response => response.json())
            .then(data => {
                document.getElementById('dashboard').innerHTML = `
                    <p>場所: ${data.city}</p>
                    <p>気温: ${data.temp}°C</p>
                    <p>湿度: ${data.humidity}%</p>
                    <p>状況: ${data.description}</p>
                `;
            });
    </script>
</body>
</html>
```

## 4. デプロイ手順

1. **ファイルの準備**: `index.html` と `data.json` を同一フォルダに配置します。
2. **Cloudflare Dropにアクセス**: [Cloudflare Drop](https://drop.cloudflare.com/) のサイトを開きます。
3. **アップロード**: フォルダ全体をブラウザ上にドラッグ＆ドロップします。
4. **公開**: 自動的に生成されたURLが発行されます。そのURLを共有するだけで、世界中からダッシュボードにアクセス可能です。

※ 定期的なデータ更新が必要な場合は、GitHub Actions等でPythonスクリプトを定期実行し、結果をデプロイ先に反映させる構成が推奨されます。

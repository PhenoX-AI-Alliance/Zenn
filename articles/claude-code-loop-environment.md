地球環境モニタリングデータの異常検知を自動化するため、Claude CodeのPlanモードを活用した開発手法を解説します。

### 手法
1. **Planモードの定義**: 観測データ（CSV）の読み込み、統計的異常検知（Z-score法）、および警告通知のパイプラインを設計。
2. **自動生成**: Claude Codeを使用して、データクレンジングから異常検知ロジックまでをモジュール化。
3. **検証**: 過去の気象データを用いて、閾値を超えた際の正確性を検証。

### 結論
AIによる開発ループを回すことで、環境データの異常をリアルタイムで検知するシステムが短時間で構築可能です。

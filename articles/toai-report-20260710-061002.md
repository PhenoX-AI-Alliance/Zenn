# 2026年7月版 Claude CodeによるAI駆動開発フロー

本記事では、Claude Codeを活用した最新のAI駆動開発（AIDD）フローを用いて、地球環境データ（気温、海面水位、CO2濃度）をリアルタイム可視化するダッシュボードを構築します。

## 1. アーキテクチャ設計
- **Frontend**: Streamlit
- **Backend**: FastAPI
- **Data Source**: e-Stat API (日本政府統計)

## 2. 実装手順
1. `claude-code` を初期化し、プロジェクト構造を定義。
2. e-Stat APIからJSONデータを取得し、Pandasで前処理。
3. 可視化エンジンを構築。

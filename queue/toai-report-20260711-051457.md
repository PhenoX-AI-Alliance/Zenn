# Cloudflare Dropで即時公開！地球環境データを可視化する静的サイト構築術

環境データの可視化は、地球環境のレジリエンスを高めるための第一歩です。本記事では、Cloudflare Dropを活用し、複雑なサーバー設定なしで環境データサイトを即時公開する手法を解説します。

## Cloudflare Dropによるデプロイ手順

1. **プロジェクト準備**: 静的サイトジェネレーターでビルドされた `dist` ディレクトリを用意します。
2. **Cloudflare Dropへのアップロード**: 
   Cloudflareのダッシュボードから「Workers & Pages」を選択し、Drop機能を使用してファイルをドラッグ＆ドロップします。
3. **即時公開**: 数秒でグローバルネットワーク上にサイトが反映されます。
